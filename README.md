# Enhanced File Explorer Application with Smart File Search in C++

## Overview
A console-based File Explorer built in C++ for Windows that allows users to list, create, delete, and search files easily.

## Features
- List all files and directories  
- Create and delete files  
- Smart Search to find files quickly  
- Simple and clean console interface
